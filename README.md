## Install

Clone the repository, and then copy the config.rasi file into ~/.config/rofi.

```sh
git clone https://github.com/BattleCh1cken/oxocarbon-rofi.git
cd oxocarbon-rofi
mkdir -p ~/.config/rofi
cp config.rasi ~/.config/rofi
```
